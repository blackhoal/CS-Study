## 1-3. 정규화
- 정규화란?
- 정규화의 목적
- 이상현상
  - 이상현상이란?
  - 이상현상의 종류
    - 삽입 이상
    - 삭제 이상
    - 갱신 이상
- 정규화 과정(3NF까지만 상세하게 / 예시를 통해 1NF ~ 3NF까지의 과정을 적으면 좋을것 같습니다 / 이후 과정은 조건 정도만 인지)
  - 1NF : 도메인이 원자값
  - 2NF : 부분적 함수 종속 제거
  - 3NF : 이행적 함수 종속 제거
  - BCNF : 결정자 중 후보키가 아닌 것 제거
  - 4NF : 다치 종속 제거
  - 5NF : 조인 종속 제거
