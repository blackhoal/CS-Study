# 2. 구현
## ① Python
```python
def solution(absolutes, signs):
    answer = 0

    for i in range(len(absolutes)) :
        if signs[i] == True :
            answer += absolutes[i]
        else :
            answer -= absolutes[i]
        
    return answer
```
