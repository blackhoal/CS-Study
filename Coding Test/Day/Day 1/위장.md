# 2. 구현
```python
def solution(clothes):
    cloth_dict = {}
    for i in clothes:
        if i[1] in cloth_dict: 
            cloth_dict[i[1]] += 1
        else: 
            cloth_dict[i[1]] = 1

    cnt = 1
    for j in cloth_dict.values() :
        cnt *= j + 1

    answer = cnt - 1

    return answer
```
